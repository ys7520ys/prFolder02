SEO:
  검색 엔진 최적화(SEO)를 잘 반영한 사이트를 의미하며,검색 엔진에서 높은 순위에 오를 가능성이 큰 사이트를 의미한다. SEO의 핵심 목표는 검색 엔진이 사이트의 내용을 이해하기 쉽게 하는 것이며, 이를 통해 검색 결과에서 더 잘 노출되도록 돕는 것이다.

  1. 메타 태그 설정:
      Title 태그:
        페이지의 제목을 명확하고 간결하게 설정하며, 주요 키워드를 포함한다.
      Meta Description:
        페이지의 내용을 요약하는 설명으로, 검색 결과에서 보여진다.
      Meta Keywords:
        검색 키워드 리스트를 제공하는 속성으로, 현재는 대부분의 검색 엔진에서 크게 영향을 주지 않지만 일부에서는 사용된다.
      
  2. 시맨틱 마크업 및 HTML 구조:
      - header, nav, main, section, footer 같은 시맨틱 태그를 사용하여 페이지의 구조를 명확하게 표현한다.
      - 헤딩 태그(h1-h6):
          페이지와 섹션의 제목에 적절한 헤딩 태그를 사용하여 검색 엔진이 페이지 계증을 이해하도록 돕는다.
  
  3. 이미지 최적화:
      - Alt 속성: 
          이미지의 설명을 적어 검색 엔진과 스크린 리더가 이미지를 이해할 수 있도록 한다.
      - 이미지 파일 크기를 줄이고 srcset을 통하여 반응형 이미지를 제공하여 로딩 속도를 개선한다.

  4. URL 구조:
      - URL은 간결하고 의미가 있어야 하며, 주요 키워드를 포함하는 것이 좋다. URL은 검색 엔진과 사용자가 페이지의 내용을 예측할 수 있게 한다.

  5. 모바일 친화성 및 반응형 디자인:
      - 모바일 사용자가 증가함에 따라 검색 엔진은 모바일 친화적인 웹사이트를 우선시한다. 반응형 디자인을 통해서 다양한 기기에서 일관된 사용자 경험을 제공한다.

  6. 로딩 속도 최적화:
      - 검색 엔진은 로딩 속도가 빠른 사이트를 선호한다. CSS, JS, 이미지 파일 크기 최적화 등을 통해서 페이지 로딩 속도를 개선하는 것이 좋다.
  
  7. 내부 링크 구조:
      - 사이트의 구조가 논리적이고 명확해야 한다. 또한, 중요 페이지로 연결되는 링크가 있어야 하고, 적절한 앵커 텍스트를 사용하여 사용자와 검색 엔진이 링크의 목적을 이해할 수 있도록 해야 한다.
